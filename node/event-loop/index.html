    # 浏览器eventLoop

    • 微任务

      promise.then、MutationObserver（监听dom变化，等待本轮dom变化完成后会执行）、process.nextTick。

    • 宏任务

      script、ajax、事件、requestFrameAnimation、setTimeout、setInterval、setImmediate（ie）、I/O操作、UI rendering。

    js是单线程的，当script执行完毕后，会先将微任务队列清空，清空后会执行宏任务；宏任务执行完毕后再次清空微任务，无限循环，直到浏览器关闭销毁。

    # Node事件环

    给每个宏任务都配置一个队列 timer poll check(每个执行完后清空一次微任务)
