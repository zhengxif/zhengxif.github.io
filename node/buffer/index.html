    # 对进制、位、字节、编码的理解

    • 进制 进位计数制，是认为定义的带进位的计数方法，对于任何一种进制--X进制，就是表示每一位置上的数运算时都是逢X进一位。计算机中所有的数据都被存储位一连串的二进制信息（0和1）。为了更加方便的表达计算机的存储的数字，计算机 使用人员使用十六进制来记录，便于书写、记录。 常用的进制包括二进制（01010011）、八进制（123）、十进制（53）、十六进制（83）

    8位二进制数字可以转换成2位十六进制数

    • 位（bit） 位（bit）是电子计算机中最小的数据单位，每一位的状态只能是0或1。bit表示字，一个二进制位；byte表示字节，1字节 = 8bit(字)
    • 字节（byte） 8个二进制构成1个字节（byte），它是存储空间的基本计量单位。1个字节可以存储1个英文字母或者半个汉子，换句话说，1个汉字占据2个字节的存储空间。 1B = 8bit、1KB = 1024B、1MB = 1024KB
    • 编码 编码是信息从一种形式或格式转换为另一种形式的过程，在计算机硬件中，编码（coding）是指代码来表示各组数据资料。通俗的讲，计算机中的二进制要变成文字、信息就要通过编码过程。

// 手动实现split
Buffer.prototype.split = function (step) {
    let arr = [];
    let offset = 0;
    let start = 0;
    step = Buffer.from(step);
    while((offset = this.indexOf(step, start)) !== -1) {
        arr.push(this.slice(start, offset));
        start = offset + step.length;
    }
    arr.push(this.slice(start));
    return arr;
}
